Course and Book Information

Book Information

This book is the textbook for the Web Interactivity and Engagement course. The irony of using Jekyll and Github to publish a course text for a course taught in the context of WordPress is not lost on me. However, this book is meant to be a collaborative effort between the author, students, and general community and having it published off of a GitHub repository helps make it collaborative. Please feel free to submit push requests about corrections and improvements to the Github repository.

Course Information

The purpose of this course is to provide the student with an understanding of the most common and practical tool of a web manager, the content management system (CMS). Using WordPress and other resources, you will gain an understanding of CMSs and how they are valuable tools for organizational content management and scalable information architecture.

The course provides a cursory understanding of PHP as a backend programming language as well as an introduction to databases and other web development best practices as applied to scalable systems.
